Instructions: choose the dimensions of the level you are creating with the x and y input fields and choose the level with the z value field. Then submit to build the grid. Then choose the Id modifier number by using the highest previous tileId in the database. DO NOT CHANGE THE ID MODIFIER NUMBER DURING A LEVEL CREATION OR YOU WILL DESTROY THE ENTIRE UNIVERSE!!!! Next choose the type of tile you want to place along with the appropriate property values and then click on the grid where you want that tile type to be. YOU MUST CLICK ALL TILES OR YOU WILL GET UNDEFINED FOR ANY UNCLICKED TILES. Enjoy!
